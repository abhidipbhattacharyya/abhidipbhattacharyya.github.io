Magnifier

I am using an Apple macbook. Hence to magnify screen I have used the "zoom" functionality provided by apple. How to use this facility can base found here. I think it is a nice tool for helping people with visual impairments. But i think this system has following limitations

  • The system is limited by the capabities of the web site. For example here I am giving an image from newyork times. In normal vision an image is almost clear. Now when I used the zoom on the image the image get pixelated. This is actually because the image is of very low quality. Hence using magnifier is not that helpful in this case. For example in this case I can not really see the face of the man pelting snow ball.

  • No text wrapping. I am not sure who should take care of this part- websites designer or the magnifier software developers. No wrapping of screen makes the user scroll horizontally the screen. This is super inconvanient as per me.

  • The left image is with magnifier this time. I just clicked the "add to cart button". And then the screen became gray with inactivating all elements. I really had no idea what was going on untill I off the magnifier. So the appearence of alert box/ dialog box is with respet to the web page. It should be relative to the user focus on the page.

What would I prefer

It will be really helpful instead of magnifing the whole screen if it magnifies the particular area where the user is focusing. So user can have an idea about the over all webpage while focusing on a particular portion. Moreover if the text can be wrapped in the magnified window the content will be more readable.

For magnifing images I would say - amazon website has nice implementation of magification for product images. So when user hover over the image it magnifies the product image and the whole image is contained in a single window. No need to scroll horizontally to see the images. However it may depend on the visual capabities of the users.

What have I done for this assignment

  • I have created a sepate site for low vision.
  • I have made the background black and foreground either white or yellow to give sufficient contrast.
  • Fot buttons and text box I made the opposite. So the main background is black, buttons' background is white-ish but text is black. That makes the button and text box noticable and readable.
  • Font-size of text has been increased to make it more readable.
