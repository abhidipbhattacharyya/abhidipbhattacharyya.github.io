About this app

This is a basic calculator app. You can click buttons to perform calculations. However it has some special features. Kindly read the instructions before use.

Instructions for using the app

It has following modes for work
  • Basic Mode- In this mode user can select any button and click it using mouse. Or user can select the text area. Once the cursor is at the text area user can give numbers from keyboard.
  • Arrow Key and Enter- User can use any of the arrow keys (left or right) and enter key of the physical key board of their computer to operate the calculator. For example only right arrow and enter. Selection will start from 7. On pressing right arrow selection will move right of the current seletion. If the current seletion is at end of a row upon pressing right arrow selection will move to the next row. To click a button hit enter. Similarly using only left key user can navigate all the buttons.

    However using up/ down arrow user can navigate only in a particular column of the buttons.

    Arrow keys functionality gives the calculator multiple modalities. For example-

    1. Two keys- User can use only right (left) arrow and enter to use this calculator
    2. Three Keys- User can use right (left) arrow, enter and any other arrow keys(up, down, left) to use this calculator
    3. Four Keys- User can use right key, left key, enter key and any of the other directional key(up, down) to navigate this calculator
    4. Five Keys- User can user all the four arrow keys and enter to naviagate this calculator.
  • Automatic Scanning (One Key Mode)- In this mode user can use only one key Enter to click the button. The system will automatically naviagete through buttons. User just need to hit enter when the desired button is selected. To start automatic scanning mode press 'S' in your keyboard. To stop automatic scanning press 'S' again. User can use any of the arrow keys during automatic scanning.
